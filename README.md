# 🐍 Python Basics Project (5-Day Learning Program)

This project is a simple Python program built step-by-step over 5 days to understand the fundamentals of Python programming, including user input, variables, and formatted output.

---

## 📅 Project Timeline

### ✅ Day 1: Setup & First Program
- Installed Python
- Created the first `.py` file
- Wrote a basic "Hello, World!" program

### ✅ Day 2: Add User Input
- Introduced variables
- Used `input()` to take user input

### ✅ Day 3: Welcome Message
- Used f-strings for formatted output
- Displayed personalized messages

### ✅ Day 4: Enhance Program
- Added multiple user questions
- Improved formatting and readability

### ✅ Day 5: Finalize & Test
- Fixed errors (debugging)
- Added comments for clarity
- Final testing of the program

---

## 🚀 Features

- Takes user input (name, preferences, etc.)
- Displays personalized messages
- Beginner-friendly structure
- Clean and readable code
